## 📚 Topics Covered
- `if`, `else`, `elif` control structures
- Numeric comparisons: `-eq`, `-ne`, `-gt`, `-lt`, `-ge`, `-le`
- String comparisons: `=`, `!=`, `-n`, `-z`
- File checks: `-e`, `-f`, `-d`, `-s`

---

## ✅ Commands Learned

### 📊 Numeric
- `-eq` (equal)
- `-ne` (not equal)
- `-gt`, `-lt` (greater/less than)

### 🧵 String
- `=`, `!=` – compare strings
- `-n` – not empty
- `-z` – is empty

### 📁 File
- `-e` – file exists
- `-f` – regular file
- `-d` – directory
- `-s` – not empty
